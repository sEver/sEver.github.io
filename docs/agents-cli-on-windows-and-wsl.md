Google Agents CLI requires Python 3.11+, Node and `uv`. 
- With `pyenv` installed Python 3.11 by:
  - `pyenv install 3.11`
  - `pyenv global 3.11`
- Then installed the `uv`
  - `pip install uv`
- And with it, installed Google Agents CLI
  - `uvx google-agents-cli setup`
